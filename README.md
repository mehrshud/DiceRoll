
# Dice Roll Game
A simple dice roll game with multiplayer mode.

## How to Play
1. Run the game by executing `python dice_roll.py`.
2. Enter the number of players.
3. Enter each player's name.
4. Press Enter to roll the dice.
5. The player with the highest score at the end of the game wins.

## Features
* Multiplayer mode
* Random dice roll
* Score tracking
* Winner declaration
